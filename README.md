# 法人化シミュレーター

> 個人事業主と法人化後の手取りを比較し、法人化の有利不利を判定するWebシミュレーションツール  
> 対象バージョン：v1.0（2025年度税制・保険料率対応）

---

## 概要

本ツールは、個人事業主が法人化を検討する際に、以下の3軸で比較・判定を行います。

| 比較軸 | 内容 |
|--------|------|
| 手取り差額 | 個人手取り vs 法人（役員手取り＋配当後留保－維持コスト） |
| 5年累計差額 | 年間差額 × 5年 |
| 実効税率 | 粗利に対する総コスト（税・社保・維持コスト）の割合 |

バックエンドとの通信は一切行わず、すべての計算はブラウザ内で完結します。

---

## 機能

### 入力モード

| モード | 対象 | 入力項目数 |
|--------|------|-----------|
| かんたんモード | 一般ユーザー | 最小限（7項目） |
| 詳細モード | 税理士・FP | 全項目（15項目以上） |

#### かんたんモード（固定値）
- 青色申告特別控除：65万円（e-Tax想定）
- 小規模企業共済：84万円（満額）
- iDeCo（個人）：81万円（満額）
- 年間維持コスト：60万円
- 法人iDeCo：0円

#### 詳細モードの追加入力項目
- 青色申告特別控除（10 / 55 / 65万円）
- 小規模企業共済（オン/オフ ＋ 金額設定）
- iDeCo個人・役員（オン/オフ ＋ 金額設定）
- 法人追加経費（社宅・社用車・生命保険等）
- 年間維持コスト（顧問料・登記更新等）

### 計算内容

#### 個人事業主
- 個人事業税（5%）
- 国民健康保険料（上限104万円）
- 国民年金（21.012万円/年、2025年度）
- 所得税（累進課税 ＋ 復興特別所得税×1.021）
- 住民税（課税所得×10% ＋ 均等割0.5万円）
- 消費税（簡易課税方式）

#### 法人化後
- 法人税等（〜800万円：26.6%、超過：36.2%）
- 法人住民税均等割（7万円）
- 社会保険料（協会けんぽ東京・2025年度、健保9.91% ＋ 厚年18.3%）
- 役員給与所得控除・役員所得税・住民税
- 配当課税（申告分離課税：20.315%）
- 設立費用（株式会社25万円 / 合同会社10万円、5年償却）

### 表示機能
- **判定バナー**：法人有利 / 個人有利 / ほぼ同等の3段階判定（年間差額・5年累計差額を表示）
- **手取り比較カード**：個人・法人の年間手取り・実効税率を並べて比較
- **詳細内訳**：税金・社保の項目別内訳（個人・法人それぞれ）
- **損益分岐点グラフ**：売上300〜5,000万円の範囲で手取りをプロット、交差点を自動検出
- **PDF出力**：計算明細込みのレポートをブラウザ内プレビュー後にA4形式でPDF保存

### レイアウト
- デスクトップ（lg以上）：入力フォーム（左）・結果（右）の2ペイン分割スクロール
- モバイル：縦積みの単一カラムレイアウト

## デプロイ

`main`ブランチへのpushで GitHub Actions が自動的にビルド・GitHub Pages へデプロイします。

### GitHub Pages の設定手順
1. リポジトリの **Settings > Pages** を開く
2. **Source** を `GitHub Actions` に設定する
3. `main`ブランチにpushすると自動デプロイ

## 免責事項

- 本ツールは概算であり、実際の税額は個別の状況により異なります。
- 申告・法人化判断の前に必ず税理士にご相談ください。
- 社会保険料は協会けんぽ（東京・2025年度）の料率を使用しています。
- 消費税は簡易課税方式で計算しています（原則課税は対象外）。

## 既知の制約・今後の拡張候補

| 項目 | 現状 | 拡張候補 |
|------|------|---------|
| 消費税計算 | 簡易課税のみ | 原則課税との比較 |
| 社会保険料 | 協会けんぽ（東京）固定 | 都道府県別対応 |
| 役員報酬最適化 | 手動設定のみ | 最適報酬の自動提案 |
| 複数年シミュレーション | 単年のみ | 5〜10年キャッシュフロー |
| 事業税率 | 一律5%で簡略化 | 業種別税率対応 |
